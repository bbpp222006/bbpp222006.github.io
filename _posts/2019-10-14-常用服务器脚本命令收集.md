2. ssr安装
`wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/ssr.sh && chmod +x ssr.sh && sudo bash ssr.sh`

4. 服务器贴吧签到重启
```
service docker start
docker-compose up -d
```
