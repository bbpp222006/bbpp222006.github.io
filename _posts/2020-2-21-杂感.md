# 前言
本博主眼界有限,文笔稀烂，博文不可避免的带有私货，权当个人牢骚，所涉及到的事件如有雷同，不胜荣幸。 

# 考了20多年的试，我得到了什么？
以考试作为线索,依据时间轴逐一回忆     

记事以来，生活中好像就有数学，英语这些东西了，记不清是小学还是学前班的时候就在学英语，时不时也有各种各样的大考小考，但时代过于久远，当初的感受与结果已经记不清，总体应该是愉快的。  
时间来到小学，这个时代的考试就已经开始有阶级分明的情况了，班上成绩差的多半会被中等的一大批小团体欺负，无论男女。（很不幸，本人也属于欺负他们的团体中的一员，在此向他们诚恳道歉。）于是小学六年期间总有人转班或者留级。
班级上好学生分为两类，一类是学习成绩好，次次班上前几名，另一类是深得老师喜爱（具体为什么我也不清楚，我也不敢问）。具体所拥有的特权包括，但不限于以下几种:  
1. 提前放学.(这个是最不能忍的)
2. 打饭时提前排队
> 关于这点我要描述一下,每天中午打饭老师会让一位学生上讲台一个个点名,先点的去打饭,最后被点的只剩下残羹冷炙. 点名的顺序由学生根据场上的学生表现自行决定,但老师就在旁边坐着. 但是大多数情况下前几个吃饭的永远都是那几位,即使他/她正在睡觉,或者聊天.
3. 免考
4. 自带洗白buff  

还有一些,不过爷忘了...不过回想起来还是能看出些端倪的, 班主任永远是班级的老大哥, 老大哥所欣赏的人物在没有自知的情况下,下面的学生会自发的去拥护他/她.   
也许在外国有部分人瞧不起书呆子, 但在中国,从小就做一个书呆子是一个最好的选择. 首先父母就会得到面子, 在家长的圈子内也能获得更高的关注度, 家长会自发的传递好学生所用的教辅, 所上的培训班等等教育资源. 而很少有人会去比较家庭环境与所拥有的人脉圈子.   
说回小学的课外, 奥数就开始出现了, 刘x数学,新东方,各种奥赛比赛. 博主也参加过不少培训班. 一节课在当年也要不少钱, 什么鸡兔同笼,进制转换计算,口算心算等等...   
同样,每场小测验按成绩排名, 越高的坐越前面, 考的差的坐后面.   
当年我听不懂, 又刚逢psp热潮, 每次考试就交白卷. 导致每次都坐最后面, (上课掏出psp开始怪猎的时候总会吸引周围一群同道中人的目光.)虽然是学渣, 但是总是会听到一些神童的传言, 比如谁谁谁又拿了什么什么国家级的奖, 然后家长就开始疯传,尽管雨我无瓜, 但在家长聊天的只言片语中总能感受到一丝羡慕嫉妒恨的情感.  
******
时间节点来到了初中, 这时的竞争已经渐入佳境, 从进好学校的好班需要多少后门关系就能看出来
> 亲身经历:初一转学校时的考试,题目巨难,结果前面考满分的神童没有进,反而是一个没及格的成功转校(猜猜是谁?)...  

初中班级种类的学生数量呈金字塔结构,竞赛班 -> 火箭班 ->  高分班 -> 普通班  
博主是普通班上的一员,初一初二成绩一般,到了初三物理化学的比重开始加大,成绩才慢慢开始有起色.  
此时,中考就开始像一座小山慢慢堆积在心脏上,每离初三毕业近一点重量就增加一倍, 从那时开始变落下了考试前肚子疼,恶心想吐的毛病.  
说到课外培训, 当初lol等游戏层出不穷, 小伙伴们天天开黑, 但只有我总是玩一半被拉去上辅导班, 要不就是晚上限时, 过点拉闸(导致我的等级一直远远落后于他们,最后导致渐渐疏远了...)  
辅导班全家桶, 语数外理化,换老师换地点换噱头.记忆比较模糊, 只有或刺眼或昏暗的灯光, 仿佛时停一般的手表, 白卡卡的讲义, 破烂的书包与笔袋. 不变的只有中考这一个东西.  
整个初中最清晰最开心的记忆莫过于和小伙伴翘课去黑网吧上网, 上课偷偷看小说漫画了.   
初中所谓的元调考过全校前十,当初兴奋了一整晚,但是也就那一晚了.  
中考发挥一般般, 进了一所比较偏远,但是风评以竞赛闻名的高中. 

******
高中了,班级分层仍然和初中类似的金字塔结构,不过这次博主在高分班.
> 初中升高中的暑假,高中举行了一次集训,集训最后进行了分班考试. 最后考试的时候由于一碗过期泡面,博主在教室里面吐了一地. (虽然这个考试很难,博主这个菜鸡啥都不会写, 但是可能影响了其他同学的发挥, 这里道个歉)

凭借在物化生方面的天赋(或者直觉?),高中一直苟在班上前十, 甚至一度苟进年级前十,高考甚至苟进985  
但这段回忆一直环绕着令人不适的感觉.  因为面对的是高考, 平常考试的频率与强度都大大高于以前. 每个周末都有一次单科的全校排名考试, 考完后全校公布结果 并在下一次考试时根据排名划分考场.(这段时间我的紧张毛病大大恶化,考试前必须去厕所一次, 到了邻近高考的末期甚至出现了口水强迫症的状况,**忠告:各位看官请不要去查什么叫做口水强迫症,因为不排除你了解这个病之后也会犯的情况,只需知道很难受就行了**)  
前俩个考场的高分班人不多, 都是竞赛班与火箭班的面孔, 虽然我不知道他们名字,他们却好像已经熟悉我了...  
课外培训沿袭了初中的作风, 只不过频率更高, 学生座位的密度更高, 上课所在的位置更破而已.  
高中三年似乎浑浑噩噩, 高强度的学习之下只能看看小说,动漫满足一下对美好生活的向往, 反而玩游戏的时间变少了.  
终于到了高考, 很奇怪的是那几天的感受已经完全无法回忆起来了, 似乎从未经历过一样, 甚至比日常的记忆还要遥远.    
18年来的奋斗到头来也只剩一张录取通知书, 班上的同学八仙过海,各奔东西, 也头一次认识到结果比过程的重要

******
到了大学, 全新的生活开始了.  
大一把以前感兴趣的东西玩了个遍, 贴吧, 补番, 小说, 游戏.... 渐渐的不再满足于此.  
大二开始研究了许多自己感兴趣的东西(俗称跳坑), 焊接电路, 水卡破解(nfc安全), 编程, 网络安全(脚本小子),机器学习, ccna, 路由器, 图拉丁捡垃圾, 薅羊毛.... 虽然大多数都~~无疾而终~~, 但是相比于课程里学的那些形而上学,与时代严重脱轨的知识(并非全部,比如模电数电这类还是很有用的), 我还是更喜欢这些看得见摸得着的成就感.  
宿舍的风气就这么歪了起来, 就像没有阳光照耀的森林角落,几棵长得奇形怪状的歪脖子树放飞自我的成长着.  
三年过去了,都没保上研究生(笑), 但是平时的成绩也不算差, 离保研的分数也就差0点几分.  
唯一的加分项还是一次假期,一个外校的高中同学拉着我打美赛, 出于好奇与兄弟情分,花了3天搞了个小奖.  

回想起来,似乎别人的大学生活充斥着各类比赛,奖状,社团,专利,论文, 群里面也总是在和老师,辅导员等人互动, 而我只是一直活在自己的小圈子里自娱自乐.  

问我后不后悔?  
后悔, 没有跟着大佬走考各种证,没有找圈子一起分享期末考试题目, 没有发论文,没有去研究各种专利, 没有去竞选各种奖学金,也没有去打各种比赛, 结果落到今天这种境地.  
不后悔, 我认识了许多志同道合的朋友, 学到了许多感兴趣的知识, 甚至阴差阳错的认识了几位导师.  

##  小结
站在我的立场,我可以说我不屑于功利的去争取机会, 但是高考我已经功利的踩着90%没有家庭条件,没有天赋或者运气的考生考上了985, 所以有x子立牌坊的嫌疑  
站在普通大学生的立场可以说世道好轮回,博主也终于沦落到这个地步  
站在博主上面的那些大佬的立场.... 可能并不会在意博主这些炮灰...  

那么究竟哪里有问题呢?  幸好博主沦落为了炮灰, 才有机会好好的研究马哲.  
在学习政治之前, 马哲,毛概等在我的印象里就是新闻联播, 党八股与~~青年大学习~~  
学完政治之后,曾经有位老师问我:你是更信了还是更不信了?  
当时我笑了笑无法回答, 因为这个目标实在是太遥远,太美好了.  就拿一句话来说
>人的全面发展是共产主义的基本特征  

当初也许无法明白这句话的含义,所以不懂它的威力. 上大学后看看周围似乎懂了些什么.
