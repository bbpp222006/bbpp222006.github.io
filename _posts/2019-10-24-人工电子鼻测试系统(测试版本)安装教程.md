## 软件环境配置
python的各种依赖包
现成算法只用到了
1. sklearn
2. matplotlib

接下来是操作流程
按下win+r，输入cmd打开命令行窗口  
复制粘贴以下内容   
**逐行输入，等安装完了再输下一条**
```
pip install matplotlib -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install scikit-learn -i https://pypi.tuna.tsinghua.edu.cn/simple
```
